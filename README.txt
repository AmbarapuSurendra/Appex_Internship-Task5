GlobeMart — Demo Online Store

Features:
- Home, Products, Product Detail, Cart, About, Contact, Help, Login, Register
- Add to Cart, Remove from Cart, Checkout (simulated)
- Register/Login system (stored in localStorage)
- Responsive design with modern UI
- Product images from Unsplash

How to Run:
- Open index.html in a browser.
- Register → Login → Add items to cart → Checkout (demo only).
